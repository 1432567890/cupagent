# Skill: slang

## Словарь сленга

пользователи пишут сленгом — понимай и отвечай по сути, не повторяй их слова

| сленг | значение |
|---|---|
| флор / floor | минимальная цена (floor price) |
| гифт / gift | подарок |
| подарки | подарки в Telegram (Telegram Gifts) |
| p2p | прямая продажа между пользователями (peer-to-peer) |
| трейд / trade | обмен подарками |
| лор / lore | предыстория/контекст чего-либо |
| дроп / drop | выпуск новой коллекции |
| вайп / wipe | удаление/сброс |
| скам / scam | мошенничество |
| rugged / rug | проект бросили владельцы |
| холдер / holder | держатель чего-либо |
| стейк / stake | заморозка активов за небольшую оплату |
| флип / флипать / флиппинг | купить подарок за дешево и перепродать дорого (за короткий срок) |
| саплай / supply | тираж коллекции подарков (сколько всего выпущено) |

не копируй сленг юзера — отвечай нормально, коротко

## Казино / скам

вирус, гифтап (giftup), гифтсдабл (giftsdouble), rolls, balls — это казино. потеряете там деньги

## Фоны подарков

Black / Onyx Black — самые ценные фоны у коллекционеров

## Улучшенные и неулучшенные подарки

- **улучшенные** — подарки с моделью, фоном, номером и ссылкой t.me/nft. лимитированные. например Lunar Snake, Plush Pepe, Scared Cat
- **неулучшенные** — лимитированные подарки, которые пока не получили улучшения. у них нет номера, нет модели/фона, нет ссылки t.me/nft, не определяются монохромом. например REDO, Coffin
- **факт:** все лимитированные подарки в итоге получат улучшения
- **крафты** — когда можно несколько подарков поставить на кон и получить один скрафченный (объединённый) подарок
