# Skill: market-history

## Назначение

История цен и кросс-маркет агрегация через Moomin Market API.

Два инструмента:
- `get_market_snapshot` — актуальные цены по маркетам + прямой floor (direct_floor)
- `get_price_history` — OHLC-свечи с компактной сводкой (summary) + рядом точек закрытия (series)

## Когда вызывать

### `get_market_snapshot`

Юзер спрашивает актуальные цены по маркетам: «сколько стоит на всех маркетах», «сравни цены», «где дешевле». Подходит как альтернатива `get_floor_prices` — даёт кросс-маркет сравнение одной коллекции за один вызов.

Параметры:
- `collection` (обязательный) — имя или слаг: "Artisan Brick", "artisanbrick", "Scared Cat"

### `get_price_history`

Юзер спрашивает про тренд/динамику: «как менялась цена», «росла ли», «упала ли», «что было неделю назад», «динамика за месяц», «тренд коллекции».

Параметры:
- `collection` (обязательный) — имя или слаг
- `market` (по умолчанию mrkt) — маркет для истории
- `interval` — 5m / 1h / 1d (по умолчанию 1d)
- `days` — сколько дней (по умолчанию 7)

## Как читать ответ

### snapshot

Возвращает `{collection, slug, quote_asset, prices: [{market, price, observed_at}], direct_floor: {market, price}}`.
Цены в TON (= GRAM 1:1). `direct_floor` — минимальная цена из прямых маркетов (не агрегированная).

### history

Возвращает компактную сводку, НЕ сырой дамп свечей:
- `summary`: open, high, low, close, change_pct (%), direction (up/down/flat)
- `series`: 12 прореженных точек закрытия [{t, close}] — по ним можно описать форму тренда
- `bars`: общее количество свечей в периоде

`direction` — "up" при росте >2%, "down" при падении >2%, иначе "flat". `change_pct` — процент изменения от open до close.

Опиши тренд по summary (направление + процент), при необходимости добавь контекст из series (ровный рост, падение с отскоком, флэт и т.д.).

## Правила

- Цены в TON (= GRAM 1:1) — пиши как GRAM в ответе
- Не выдумывай тренды — только из ответа инструмента
- Если `bars` = 0 — данных за этот период нет, скажи прямо
- Юзер указал конкретный маркет → передавай `market`
- Не вызывай историю если юзер просто спросил «сколько стоит» — для этого `get_floor_prices` или `get_market_snapshot`
