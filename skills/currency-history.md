# Skill: currency-history

## Назначение

История курса криптовалют и фиата (OHLC-свечи + сводка тренда).

Один инструмент:
- `get_currency_history` — динамика курса крипты или фиата за период

## Когда вызывать

Юзер спрашивает про тренд/динамику **курса валюты/крипты**: «как менялся
биток», «история курса тонкоина», «как рос доллар к рублю за месяц»,
«график евро», «что было с криптой за неделю», «рос ли ETH».

Параметры:
- `from` (обязательный) — исходная валюта/крипта: btc, gram, ton, eth,
  usd, rub, eur и т.д. Сленг ок: биток→btc, грам→gram, руб→rub.
- `to` (обязательный) — целевая для котировки. from=btc to=rub → история
  битка в рублях. from=usd to=rub → история курса доллара.
- `interval` — 5m / 1h / 1d (по умолчанию 1d). Фиат всегда дневной,
  поле игнорируется.
- `days` — сколько дней (по умолчанию 7 для крипты, 30 для фиата).

## Что покрывает

| Пара | Источник |
|------|----------|
| crypto → crypto/USD/USDT | Binance klines |
| crypto → fiat | Binance klines × USD→fiat daily |
| fiat → fiat (USD/EUR/GBP/CNY/...) | Frankfurter (ЕЦБ) |
| fiat → fiat с рублём/гривной/тенге/лари/белруб | ЦБ РФ archive |

RUB, UAH, KZT, GEL, BYN и пр. ЦБ-шные валюты идут через архив ЦБ РФ —
другого открытого источника истории по ним нет (ЕЦБ убрал RUB в 2022).

## Как читать ответ

Возвращает компактную сводку, НЕ сырой дамп свечей:
- `summary`: open, high, low, close, change_pct (%), direction (up/down/flat)
- `series`: 12 прореженных точек закрытия [{t, close}] — по ним можно
  описать форму тренда
- `bars`: общее количество свечей в периоде
- `source`: какой источник ответил (binance / frankfurter / cbr)

`direction` — "up" при росте >2%, "down" при падении >2%, иначе "flat".
`change_pct` — процент изменения от open до close.

Опиши тренд по summary (направление + процент), при необходимости добавь
контекст из series (ровный рост, падение с отскоком, флэт и т.д.).

## Правила

- Крипту пиши тикером в верхнем регистре: BTC, GRAM, TON, ETH, USDT.
- Фиат — словами: рублей, долларов, евро, тенге, гривен.
- Не выдумывай тренды — только из ответа инструмента.
- Если `bars` = 0 — данных за этот период нет, скажи прямо.
- Не вызывай историю если юзер просто спросил «сколько стоит 1 BTC в
  рублях» — для этого `convert_currency`.
- Фиат всегда дневной — не обещай внутридневную динамику по рублю/евро.
