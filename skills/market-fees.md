# Skill: market-fees

## Назначение

Комиссии маркетплейсов подарков. Когда юзер спрашивает про комиссию
маркета/при продаже/при покупке/сколько забирает площадка — используй эти
данные. **НЕ выдумывай** проценты для маркетов, которых здесь нет.

## Комиссии

| Маркет    | Комиссия                                          |
|-----------|---------------------------------------------------|
| Grapes    | **5%**, но для некоторых коллекций иногда **1%**  |
| Tonnel    | **0.5%**                                          |
| MRKT      | **2%**                                            |
| GetGems   | **2%**                                            |
| Portals   | **2%**                                            |

## Правила

- Говори только то, что в таблице. Не добавляй «обычно», «примерно».
- Если юзер спрашивает комиссию конкретной коллекции на Grapes — честно
  скажи, что точную ставку для конкретной коллекции не знаешь: по умолчанию
  5%, но для части коллекций действует 1%, и какой именно — нужно проверять
  на самом маркете.
- Не путай комиссию маркетплейса с газом TON/ончейн-сетью — это разные вещи.
