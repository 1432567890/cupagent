## План реализации

### 1. Создать `skills/slang.md` — сленг + люди + маркеты + базовые тиры номеров
Новый скилл с:
- **Словарь сленга**: flore/floor, gift/gifts, NFT/present, p2p, trade, и т.д. (короткий, для навигации модели)
- **Люди комьюнити**: перо (autoGifts, история с 4 июля 2025), нон (see.tg), софа скаллива (podedala_bot), фриман (tonnel), роксман (portals, major), луча (portals), тайлер (mrkt), игнорант/старков (giftchanges API), зебатук (scared cats)
- **Маркетплейсы**: грейпс (без казино), мркт (казино, осуждаемо), порталс (как мркт), getgems (казино lucky buy)
- **Казино/скам**: вирус/гифтап, гифтсдабл, rolls, balls — потеряете деньги
- **Фоны**: Black/Onyx Black ценятся у коллекционеров
- **Номера подарков**: краткий обзор тиров (S/A/B/C/D) — подробности в отдельном skill

### 2. Создать `skills/gift-numbers.md` — полная классификация номеров
Полный справочник тиров из прикреплённого текста:
- Tier-S: одинаковые (7, 77, 777...)
- Tier-A: круглые, удачные (китайские), популярные, 10-99
- Tier-B: ступеньки, ступеньки вниз, одинаковые ±1, пары подряд, тройки подряд, ступени-зеркала, зеркала вниз, ступени зеркала
- Tier-C: чередование, пары групп, группы-зеркала, годы, амбиграммы, 100-999, зеркала, повтор+хвост, пары лесенкой, обратные пары лесенкой
- Tier-D: двойной шаг, двойной шаг вниз, повтор шагом, повтор вниз, две группы, групповые, XY+нули, XYZ+нули, палиндромы (5 и 6 цифр), периодические блоки, фибоначчи

### 3. Обновить `skills/instruction.md`
- Добавить секции Skills для `slang` и `gift-numbers` (краткие описания когда использовать)
- Добавить упоминание GiftWiki для lookup тиров номеров

### 4. Обновить `bot/handlers/guest_chat.py` — inline edit как в chat.py
Заменить `answerGuestQuery` на:
1. Отправить обычное сообщение `<i>думаю...</i>` как reply
2. Запустить typing action loop (как в chat.py)
3. Вызвать LLM с on_tool_call callback (edit на "получаю данные...")
4. Edit status message на финальный ответ с post-processing (ghost format, emoji)
5. Fallback: если edit не удался → delete + send new

Удалить `_build_result()` helper (больше не нужен) и `InlineQueryResultArticle` импорты. Добавить `_typing_action_loop` из chat.py (вынести в _chat_core.py для переиспользования или дублировать — лучше вынести).
